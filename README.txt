HƯỚNG DẪN ĐĂNG WEB CÁ NHÂN "TIACHOPVANG" LÊN GITHUB PAGES

1. Truy cập https://github.com và đăng nhập (hoặc tạo tài khoản mới).
2. Tạo một repository mới, đặt tên: tiachopvang
3. Upload toàn bộ thư mục website-tiachopvang lên (gồm index.html, style.css, assets, README.txt)
4. Vào Settings > Pages > Chọn branch “main” và folder “/ (root)” > Save
5. Sau vài phút, truy cập web của bạn tại:
   https://[tên_github].github.io/tiachopvang
